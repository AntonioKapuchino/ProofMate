О ProofMate

ProofMate — это инновационный веб-сервис для автоматической проверки решений задач по линейной алгебре, разработанный специально для преподавателей и студентов, работающих с Jupyter-блокнотами.

Наши передовые алгоритмы проверки анализируют пошаговые решения в формате .ipynb, предоставляя подробные отзывы, обнаружение ошибок и предлагаемые исправления для матриц, векторных пространств, линейных преобразований и многого другого.
